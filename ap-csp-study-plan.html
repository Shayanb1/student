💡 Exam Day Tips

  • Use the AP pseudocode reference sheet — don't memorize syntax cold
  • For MCQ, eliminate 2 wrong answers first
  • For written response: state the purpose, then explain the algorithm step-by-step
  • Flag tricky questions and come back — don't get stuck
  • RANDOM(a, b) is inclusive of both a and b in AP CSP
  • List indexing starts at 1, not 0
  • Every IP address is unique; domain names use DNS to resolve
  • Sleep 8+ hrs the night before — this is not optional
